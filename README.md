# Gestor de Lecturas

Este proyecto permite gestionar lecturas (libros, artículos, notas y resúmenes).

## Objetivos
- Registrar libros y lecturas.
- Crear notas y resúmenes.
- Llevar seguimiento del progreso.
- Almacenar información de manera organizada.

## Estructura del Proyecto
- src/: Código fuente
- docs/: Documentación
- tests/: Pruebas
- BITACORA.md: Registro de actividades del proyecto

## Flujo Git
- Rama principal: main
- Ramas de trabajo: feature/*, fix/*, docs/*
